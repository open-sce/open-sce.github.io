Openness

We believe in the power of open-source and its ability to drive positive change. All projects and contributions within this community are released under an open-source license.

Collaboration

We encourage collaboration and value the input of diverse perspectives. Collaboration includes not only developers but also designers, testers, documenters, and users.

Transparency

We promote transparency in decision-making processes, project management, and governance. Discussions, roadmaps, and decisions are accessible to all members.

Study Migration Tool

Study Migration Tool has been added to Open SCE. Study Migration Tool is a Plotly-Dash Python application developed by Achieve Intelligence that generates visualisations of study milestone activity for analysis and migration scheduling. The full code of this package is available in Open SCE (GitHub)
