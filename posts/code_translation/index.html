Code Translation Toolset

Aug 8, 2023

A number of assets for automatic language translation using generative AI has been added to Open SCE.

Domino Data Lab released a code translation toolset, which currently includes:

  • CodeConversion.ipynb — Uses the OpenAI API to translate source code from/to a number of different programming languages (e.g. R, Python, SAS)
  • CodeConversionMagic.ipynb — Creates a sasconvert Jupyter magic command for translating SAS code into Python code

The full code of this package is available in Open SCE (GitHub)
